# fis3网站组件化开发框架

## 前言

本文是为了帮助新加入团队，进行WebPC前端开发的小伙伴快速上手的帮助指南。

## 本框架目标

- 规范团队开发模式
- 更好的分工职能，避免互相交叉依赖
- 符合主流Web开发编码标准

## 编码规范

[前往阅读编码规范](https://itheima2017.gitbooks.io/front-end-standards/)

## 技术栈

fis3 + gulp + scss + ejs
